インジケーターの横に表示されてる秒数カウンタについては、mm:ss/mm:ss（per%）の形式で表示するように変更したい
カウント時間/目標時間（現在のパーセント%）が良いです。

あと、作業中/休憩中/長期休憩中の表示もアニメーションがあると良いです。
以下のような感じが良いです。
- 作業中：人間が走ってるアニメーション
- 休憩中：人間がストレッチしてるアニメーション
- 長期休憩中：人間が寝てるアニメーション

アニメーションはターミナルの横幅もある程度使うような感じでお願いします。

全体的に以下のような表示イメージです。

```
🍅 作業中 [インジケーター] 05:23/25:00 (21%)
[アニメーション]
タスク: 書類作成
```
